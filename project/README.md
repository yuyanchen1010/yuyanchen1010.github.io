# Academic Profile Website

React + Vite academic profile website for PhD applications.

## Run Locally

```bash
npm install
npm run dev
```

## Content

Replace the placeholder profile, research interests, education, projects, news, and contact information in:

```text
src/content/siteContent.ts
```

The site automatically chooses Chinese for Chinese browsers and English otherwise. The language switch in the navigation can override it during the current session.
